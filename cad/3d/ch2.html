Elevation

1. Elevation

Stores the elevation for new objects relative to the current UCS for the

current space.

1.   Type ELEVATION at the command prompt. Command: elevation

Enter new value for ELEVANT <0.0000>: 1.00

2.   Draw two circles at the new elevation. Note that they appear to be

"floating" I unit above the ground.

2. Elevation Shortcut

1.   Type ELEV at the command prompt. Command:elev

Specify new default elevation:<1.0000>: 1

Specify new default thickness:<3.0000>: 1

2.   Draw a new line to see the elevation and thickness settings.
