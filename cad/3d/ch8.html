3 D Model Objects

1.   Wireframes

A wireframes model is a skeletal description of a 3D object. There are no surface in a wireframe

model: it consists only of points, lines, and curves that describe the edges of the object. With

AutoCAD you can create wireframe models by positioning 2D objects anywhere in 3D space.

AutoCAD also provides some 3D wireframe objects, such as 3D polylines ( that can only have a

CONTINUOUS linetype) and splines. Because each object that makes up a wireframe model must be

independently draw and positioned, this type of modeling can be the most time-consuming.

2. Surfaces

Surface modeling is more sophisticated than wireframe modeling in that it defines not only the edges

of a 3D object, but also its surfaces. The AutoCAD surface modeler defines facted surface using a

polygonal mesh. Because the faces of the mesh are planar, the mesh can only approximate curved

surfaces.

 

3. Solids

 

Solid modeling is the easiest type of 3D modeling to  use. With the AutoCAD solid modeler, you can

make 3D objects by creating basic 3D shapes: boxes, cones, cylinders, spheres, wedges, and tori (do-

nuts). You can then combine these shapes to create more complex more complex solids by joining or subtracting

them or finding their intersecting (over-lapping) volume. You can also create solids by sweeping a

2D object along a path or revolving it about an axis.
